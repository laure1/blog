strstr()函数介绍：
用于查找字符串的首次出现
语法：strstr ( string `$haystack` , mixed `$needle` [, bool `$before_needle` = FALSE ] ) : string
返回 haystack 字符串从 needle 第一次出现的位置开始到 haystack 结尾的字符串。

该函数区分大小写。如果想要不区分大小写，请使用 stristr()。
